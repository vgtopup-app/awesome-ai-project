## Projects

- [LangChain](https://github.com/langchain-ai/langchain) — Build context-aware reasoning applications with LLMs.
- [Transformers](https://github.com/huggingface/transformers) — State-of-the-art ML models for text, vision and audio.
- [Ollama](https://github.com/ollama/ollama) — Run Llama, Mistral and other LLMs locally.
- [LlamaIndex](https://github.com/run-llama/llama_index) — Data framework for LLM applications over your data.
- [vLLM](https://github.com/vllm-project/vllm) — High-throughput, memory-efficient LLM inference engine.
- [Dify](https://github.com/langgenius/dify) — Open-source LLM app development platform.
- [AutoGPT](https://github.com/Significant-Gravitas/AutoGPT) — Build and run autonomous AI agents.
- [OpenAI Cookbook](https://github.com/openai/openai-cookbook) — Examples and guides for using the OpenAI API.
